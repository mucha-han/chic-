# 이미지 토글 사이트

간단하게 화면을 클릭하면 두 개의 이미지가 번갈아가며 나타나는 토글 사이트입니다.

## ✨ 기능

- **첫 화면**: 1번 이미지가 표시됩니다
- **화면 클릭**: 2번 이미지로 전환됩니다
- **다시 클릭**: 1번 이미지로 다시 전환됩니다
- **토글 방식**: 클릭할 때마다 1번 ↔ 2번 이미지가 번갈아가며 나타납니다

## 🎨 디자인

- 전체 화면을 사용하는 심플한 디자인
- 검은색 배경
- 이미지는 화면에 맞춰 자동으로 크기 조절
- 클릭 가능한 커서 표시

## 📁 파일 구조

```
└── index.html    # 메인 HTML (모든 기능 포함)
```

## 🔧 사용 방법

1. 사이트를 열면 첫 번째 이미지가 표시됩니다
2. 화면 아무 곳이나 클릭하면 두 번째 이미지로 바뀝니다
3. 다시 클릭하면 첫 번째 이미지로 돌아갑니다
4. 계속 클릭하면 두 이미지가 번갈아가며 나타납니다

## 🔄 커스터마이징

이미지를 변경하려면 `index.html` 파일의 다음 부분을 수정하세요:

```javascript
const image1 = '첫번째이미지URL';
const image2 = '두번째이미지URL';
```
